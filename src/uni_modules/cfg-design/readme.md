## CFG Design 一个可配置的 UI 框架
```
// script
import { setTextConfigs } from '@/uni_modules/cfg-design'

setTextConfigs({
  myText: { ... },
  myText2: { ... },
})

// template
<c-text c="myText" text="myText" />
<c-text c="myText2">myText2</c-text>
```
